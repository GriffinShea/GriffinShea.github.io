Description:

SVG curling is a two-player digital version of everyone's favourite stone throwing ice sport. Two web browsers connect to a central server that coordinates the shared game state in real-time through a WebSocket connection. Backend communication logic is carefully defined to avoid deadlocks and disallow abuse from a browser. Developed in Javascript with the .NET framework, graphics are generated using the SVG (Scalable Vector Graphics) library. The game itself is not really complete, but the client-server communication is a good example of how to make a simple two player game in the browser.
